> [!warning]  
> Lo sviluppo di questo progetto è stato diviso in due repository per esigenze legate alla consegna del progetto:
> * **[Repo - MakerManager-Client]** per lo sviluppo dell'interfaccia di MakerManager
> * **[Repo - MakerManager-Server]** per la logica di back-end del server

> **Questa repository conterrà solo la logica server di MakerManager, per tutta la documentazione e i dettagli sul progetto si rimanda alla repo "principale" del progetto:**  

> [**https://gitlab.com/peppe27/MakerManager-Client**](https://gitlab.com/peppe27/MakerManager-Client)
